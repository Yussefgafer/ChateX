Spectral Mesh Documentation

Welcome to the decentralized void.

1. Mesh Networking

ChateX uses a multi-transport mesh (Bluetooth, Wi-Fi Direct, LAN) to form a self-healing network. Packets hop between nodes to reach their destination.

2. Nostr Integration

Cloud transport acts as a global bridge when local nodes are not reachable, utilizing decentralized Nostr relays (Kinds 1 & 4).

3. Security

All private messages are E2EE using ECDH (secp256k1) and AES-GCM. Broadcasts use Mesh Group Encryption.

4. Fidget Physics

The UI is alive! Use the Radar view to see real-time network topology with organic magnetic snapping.
