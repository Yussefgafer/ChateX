CHATEX PROTOCOL DOCUMENTATION

Mesh Networking

GhostMesh uses a decentralized MANET (Mobile Ad-hoc Network) architecture. Nodes communicate directly via Bluetooth, LAN, and WiFi Direct without centralized servers.

The Nostr Bridge

Our bridge acts as a gateway between the local mesh and the global cloud. It uses Nostr relays for presence and message relaying when nodes are physically distant.

Fidget Physics

The UI utilizes spring-based motion schemes. Every interaction creates a tactile, magnetic feedback loop, simulating the fluidity of the Connected Void.
